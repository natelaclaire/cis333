# Class 1 Quiz: PHP Basics

## Instructions
Choose the best answer for each question.

1. Which best describes PHP in web development?
   A. A client-side scripting language that runs in the browser
   B. A server-side scripting language that generates output like HTML
   C. A database query language used only for MySQL
   D. A markup language for styling web pages

2. Which PHP tag styles are allowed by PSR-1?
   A. `<? ... ?>` and `<% ... %>`
   B. `<?php ... ?>` and `<?= ... ?>`
   C. `<?php ... ?>` and `<? ... ?>`
   D. `<?= ... ?>` and `<script> ... </script>`

3. In a PHP-only file, why is it recommended to omit the closing `?>` tag?
   A. It is required for the file to run
   B. It enables short tags automatically
   C. It prevents accidental whitespace/output after the tag
   D. It improves PHP execution speed

4. Which statement about PHP statements is correct?
   A. Statements must end with a colon
   B. Statements must end with a semicolon
   C. Statements must end with a comma
   D. Statements do not need any terminator

5. Which option correctly creates a variable in PHP using camelCase?
   A. `total-score = 10;`
   B. `$total_score = 10;`
   C. `$TotalScore = 10;`
   D. `$totalScore = 10;`

6. Which is true about `echo` and `print`?
   A. Both are functions and require parentheses
   B. Both are language constructs, not functions
   C. Only `print` can output strings
   D. Only `echo` returns a value

7. Which operator is used for string concatenation in PHP?
   A. `+`
   B. `.`
   C. `&`
   D. `::`

8. Which combined assignment operator appends to a string variable?
   A. `+=`
   B. `-=`
   C. `.=`
   D. `*=`

9. Which statement about increment operators is correct?
   A. `$count++` increments before the value is used
   B. `++$count` increments before the value is used
   C. Both prefix and suffix always return the same value
   D. Increment operators are binary operators

10. Which is a valid way to define a constant in PHP?
    A. `$MAX_SIZE = 100;`
    B. `const("MAX_SIZE", 100);`
    C. `define("MAX_SIZE", 100);`
    D. `$define("MAX_SIZE", 100);`

## Answer Key
1. B
2. B
3. C
4. B
5. D
6. B
7. B
8. C
9. B
