# 4.4 Flow Control Using `switch` Statements

In this video, we cover `switch` statements as an alternative to long
`if...elseif...else` chains when you are matching a single value.

## Basic `switch`

Use `switch` when you want to compare one value to several fixed options.

```php
<?php
$day = 'Tuesday';

switch ($day) {
    case 'Monday':
        print 'Start of the workweek!' . PHP_EOL;
        break;
    case 'Tuesday':
        print 'It\'s Tuesday!' . PHP_EOL;
        break;
    case 'Wednesday':
        print 'Midweek already.' . PHP_EOL;
        break;
    default:
        print 'Enjoy your day!' . PHP_EOL;
}
?>
```

## `break` and `default`

Each `case` should end with `break` to avoid fall-through. The `default`
case is optional but provides a fallback when nothing matches.

```php
<?php
$grade = 'B';

switch ($grade) {
    case 'A':
        print 'Excellent!' . PHP_EOL;
        break;
    case 'B':
        print 'Good job!' . PHP_EOL;
        break;
    case 'C':
        print 'You passed.' . PHP_EOL;
        break;
    default:
        print 'Invalid grade.' . PHP_EOL;
}
?>
```

## Grouping Cases

Multiple cases can share the same block.

```php
<?php
$fruit = 'apple';

switch ($fruit) {
    case 'apple':
    case 'banana':
    case 'grape':
        print 'This is a fruit.' . PHP_EOL;
        break;
    default:
        print 'Unknown item.' . PHP_EOL;
}
?>
```

## When to Use `switch`

Prefer `switch` when you are comparing a single value against specific
options. Use `if...elseif` when conditions involve ranges or complex
expressions.

```php
<?php
$score = 85;

if ($score >= 90) {
    print 'Grade: A' . PHP_EOL;
} elseif ($score >= 80) {
    print 'Grade: B' . PHP_EOL;
} else {
    print 'Grade: C or lower.' . PHP_EOL;
}
?>
```
