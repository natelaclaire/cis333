# Class 2 Quiz: PHP Data Types

## Instructions
Choose the best answer for each question.

1. Which of the following is a scalar type in PHP?
   A. array
   B. object
   C. string
   D. callable

2. Which PHP function checks whether a value is a float?
   A. `is_double()`
   B. `is_float()`
   C. `is_decimal()`
   D. `is_number()`

3. What does `gettype($value)` return?
   A. The value converted to a string
   B. The size of the value in bytes
   C. The type name as a string
   D. A boolean indicating whether the value is set

4. Which function checks whether a string contains only digits?
   A. `is_numeric()`
   B. `ctype_digit()`
   C. `ctype_alpha()`
   D. `is_int()`

5. Which statement about NULL is correct?
   A. NULL is the same as an empty string
   B. A variable set to NULL is the same as an unset variable
   C. NULL represents the absence of a value
   D. NULL is a scalar type

6. Which operator checks both value and type?
   A. `==`
   B. `!=`
   C. `===`
   D. `<>`

7. What does this expression evaluate to?
   `"42" + 8`
   A. "428"
   B. 50
   C. 42
   D. "42 + 8"

8. Which is a correct way to cast a value to an integer?
   A. `(int) $value`
   B. `int($value)`
   C. `integer($value)`
   D. `toInt($value)`

9. Which `*val()` function converts a value to a string?
   A. `intval()`
   B. `floatval()`
   C. `strval()`
   D. `boolval()`

10. What does the spaceship operator `<=>` return when the left value is greater than the right value?
    A. -1
    B. 0
    C. 1
    D. true

## Answer Key
1. C
2. B
3. C
4. B
5. C
6. C
7. B
8. A
9. C
10. C
