# 3.7 Sprintf Formatting

In this video, we focus on `sprintf` for building formatted strings. It is ideal when you want precise control over numeric formatting, alignment, or text layout. We'll also cover a couple of other useful formatting functions.

## Basic Placeholders

`sprintf` returns a formatted string based on a format pattern. `printf` prints the formatted result directly. The placeholders that you can use in the format string are documented in the PHP manual: [sprintf](https://www.php.net/manual/en/function.sprintf.php).

```php
<?php
$name = "Ada";
$score = 93.5;

$message = sprintf("%s scored %.1f", $name, $score);
print $message . PHP_EOL;
// Ada scored 93.5

printf("%s scored %.1f", $name, $score);
// prints: Ada scored 93.5
?>
```

Common placeholders:

- `%s` string
- `%d` integer (decimal)
- `%f` floating-point

## Precision and Rounding

Use precision to control decimals for floats.

```php
<?php
$price = 19.995;

echo sprintf("$%.2f", $price) . PHP_EOL; // $20.00
?>
```

## Width and Padding

You can set a minimum width and pad with spaces or zeros.

```php
<?php
$orderId = 42;

echo sprintf("%06d", $orderId) . PHP_EOL;  // 000042
echo sprintf("%10s", "Ada") . PHP_EOL;    // right-aligned
?>
```

To left-align, add a minus sign.

```php
<?php
echo sprintf("%-10s", "Ada") . PHP_EOL; // left-aligned
?>
```

## Mixing Types Safely

Match placeholders to the value types you pass in.

```php
<?php
$name = "Ada";
$count = 3;
$total = 27.5;

$summary = sprintf("%s ordered %d items for $%.2f", $name, $count, $total) . PHP_EOL;
?>
```

## number_format for Currency and Totals

`number_format` is useful when you want readable numbers with grouped thousands and fixed decimals.

```php
<?php
$total = 1234567.5;

echo number_format($total, 2) . PHP_EOL; // 1,234,567.50
?>
```

## Quick Practice

Try formatting the same values in multiple ways and compare the results.

```php
<?php
$value = 7.3;

echo sprintf("%.0f", $value);
echo "\n";
echo sprintf("%.3f", $value);
echo "\n";
echo sprintf("%8.2f", $value);
echo "\n";
?>
```

`sprintf` gives you consistent formatting for output, logs, and templates.
