# 3.8 More String Functions

In this video, we look at more string helpers that are handy for counting, searching, and formatting text.

## Counting Substrings

`substr_count` counts how many times a substring appears.

```php
<?php
$text = "banana";

echo substr_count($text, "an") . PHP_EOL; // 2
?>
```

## Counting Characters

`count_chars` gives information about character frequency.

```php
<?php
$text = "apple";

$counts = count_chars($text, 1);
var_dump($counts);
// $counts is an array of ASCII codes to counts
?>
```

## Finding the First Match

`strstr` returns the part of a string starting at the first match. `stristr` does the same but is case-insensitive.

```php
<?php
$email = "Ada@Example.com";

$domain = strstr($email, "example");
$domainInsensitive = stristr($email, "example");
var_dump($domain);            // bool(false)
var_dump($domainInsensitive); // string(11) "Example.com"
?>
```

## Repeating Strings

`str_repeat` repeats a string a number of times.

```php
<?php
$line = str_repeat("-", 20);
echo $line . PHP_EOL; // --------------------
?>
```

## Padding Strings

`str_pad` pads a string to a minimum length. You can pad left, right, or both sides.

```php
<?php
$label = "PHP";

echo str_pad($label, 10) . PHP_EOL; // right padding by default

echo str_pad($label, 10, ".") . PHP_EOL;
echo str_pad($label, 10, ".", STR_PAD_LEFT) . PHP_EOL;
echo str_pad($label, 10, ".", STR_PAD_BOTH) . PHP_EOL;
?>
```
