# 3.6 String Functions

In this video, we survey common string functions you will use in PHP. These
are the building blocks for cleaning, searching, and formatting text.

## Length and Trimming

Use `strlen` to get the length of a string, and `trim` to remove leading and
trailing whitespace.

```php
<?php
$title = "  PHP Crash Course  ";

echo strlen($title);   // 20
$clean = trim($title);
echo strlen($clean);   // 16
?>
```

You can also use `ltrim` or `rtrim` to remove whitespace on only one side.

## Case Conversion

Use these helpers to normalize text casing.

```php
<?php
$language = "php";

echo strtoupper($language); // PHP
echo strtolower("HeLLo");   // hello
echo ucfirst("hello");      // Hello
echo ucwords("hello world"); // Hello World
echo lcfirst("Hello");     // hello
?>
```

## Finding and Slicing

`strpos` finds the position of a substring. `substr` extracts part of a string. Note that PHP is a 0-based index language, which means that the first character is at position 0, the second at position 1, and so on.

```php
<?php
$email = "ada@example.com";

$at = strpos($email, "@");
print $at . PHP_EOL;
// 3

$domain = substr($email, $at + 1);
print $domain . PHP_EOL;
// example.com
?>
```

Since `strpos` can return 0 when the match is at the beginning, you must always use strict comparison when checking for failure.

```php
<?php
$text = "hello";

if (strpos($text, "h") === false) {
    echo "Not found";
} else {
    echo "Found";
}
?>
```

## Replacing Text

Use `str_replace` to replace text in a string. Use `substr_replace` to replace part of a string by position.

```php
<?php
$sentence = "Hello, world";

$updated = str_replace("world", "PHP", $sentence);
print $updated . PHP_EOL;
// Hello, PHP

$phone = "Phone 207-555-1234";
$withColon = substr_replace($phone, ":", 5, 0);
print $withColon . PHP_EOL;
// Phone: 207-555-1234

$dept = "Department of Labour";
$deptUS = substr_replace($dept, "o", 17, 2);
print $deptUS . PHP_EOL;
// Department of Labor

$dept = "Department of Labour";
$deptUS = substr_replace($dept, "", 18, 1);
print $deptUS . PHP_EOL;
// Department of Labor
?>
```

## Unicode Note

Most of these functions work on bytes, not Unicode characters. If you need character-aware behavior for multibyte text, look at the `mb_*` string functions, which we'll discuss later this week.
